# SANITARIUM
Point & click адвенчура в мрачном сеттинге.

## Текущие задачи
Для разработки создана таблица с заданиями ([тык](https://ru.yougile.com/team/b4b139295536/Sanitarium)). В ней находятся все актуальные задачи для разработки.

## Работа с репозиторием.
Для того чтобы эффективнее разрабатывать игру, будем применять такой формат:
* Берем задачу в доске
* Создаем отдельную ветки от development исключительно для этой функции(формат названия ветки development-имя-название_функции) Пример: _development-andrey-ui_
* Реализуем функционал, коммитим
* Создаем merge реквест в ветку development.
